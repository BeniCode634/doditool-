Sobre o Dodi Tool

O Dodi Tool é um projeto foda criado pela Team DODIDEIYU, usando Python e JavaScript, com uma interface de linha de comando (CMD) que roda liso até em PCs mais fracos. É perfeito pra quem quer uma ferramenta eficiente sem precisar de um supercomputador. Desenvolvido com carinho, o Dodi Tool já tá maneiro, mas em breve vai ficar ainda mais brabo com novas funcionalidades e melhorias! Suporta scripts em Python (.py) e JavaScript (.js), trazendo versatilidade pra quem curte codar ou só quer algo prático.
